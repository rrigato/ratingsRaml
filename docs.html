Toonami Ratings API API documentation version v1

http://toonamiratings.com

/ratings

Collection of available ratings for toonami ratings.

/ratings get

get

Get a list of ratings.

get /ratings

Get a list of ratings.

  • Request
  • Response

Query Parameters

  • show: (string)

    Give a show name and this will return the first 20 rating elements

    Example:

    One Punch Man
  • saturday: (string)

    Provide the date you want toonami ratings for must be in YYYY-MM-DD format

    Example:

    2014-01-04

HTTP status code 200
